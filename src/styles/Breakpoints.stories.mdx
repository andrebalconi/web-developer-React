# Breakpoints

Breakpoints for responsive layout:

<table>
  <tr>
    <th>Extra Small (xs)</th>
    <td>Default</td>
  </tr>
  <tr>
    <th>Small (sm)</th>
    <td>>= {BreakpointSize.sm}px</td>
  </tr>
  <tr>
    <th>Medium (md)</th>
    <td>>= {BreakpointSize.md}px</td>
  </tr>
  <tr>
    <th>Large (lg)</th>
    <td>>= {BreakpointSize.lg}px</td>
  </tr>
  <tr>
    <th>Extra Large (xl)</th>
    <td>>= {BreakpointSize.xl}px</td>
  </tr>
</table>

Using in styled component:

```js
import { breakAt, BreakpointSize } from Breakpoints;

const Title = styled.h1`
    font-size: 1rem;

   ${breakAt(BreakpointSize.md)} {
       font-size: 2rem;
   }
`;
```
